# Q02v2

Question 2 Version 2

网络喷子键盘侠的输入框上已经键入了一句喷人的话，这句话含有**K**个单词，现在他的键盘上只有三个键可用
C：拷贝，可以把输入框的所有内容拷贝下来
P：粘贴，可以在输入框结尾处添加刚刚拷贝的所有内容
B：退格，可以删除一句话
请问他至少需要按多少次，可以把输入框里的话复制**N**次发送出去呢？

如果不可能，则返回{-1, ""}。

保证 N >= 0

## 样例

```
N = 0
1次：B

N = 1
0次：

N = 2， K=3
2次：CP

N = 11， K=1
8次：CPCPCPPB
```

## 分析

这是一个质因数分解的题目。
